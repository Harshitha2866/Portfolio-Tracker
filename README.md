# Portfolio Tracker

## Overview

Portfolio Tracker is a beginner-friendly Python application that helps users track their stock investments. Users can enter stock details, calculate the total investment, current portfolio value, and determine whether they have made a profit or a loss.

## Features

- Add multiple stocks
- Calculate investment amount
- Calculate current portfolio value
- Display profit or loss for each stock
- Show overall portfolio summary

## Technologies Used

- Python

## Python Concepts Used

- Variables
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements

## How to Run

1. Download the project.
2. Open the project folder.
3. Run:

```bash
python main.py
```
